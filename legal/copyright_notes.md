# Copyright Notes

This repository records TableUp IP concepts, prompts, character descriptions, and generated or commissioned visual references.

## Required Metadata For New Assets

- Asset name
- Creator or generation tool
- Date created
- Source prompt or design brief
- Usage scope
- License or ownership note
- Whether it is final, draft, or reference-only

## Do Not Store

- Third-party copyrighted characters
- Unlicensed stock images
- User private data
- App credentials or production code
