# Naming Guidelines

## Naming Principles

Names should be short, memorable, food-related, and easy to say in Chinese. English names should be direct enough for prompt engineering and international asset naming.

## File Naming

Use lowercase snake_case for folders and files.

Examples:

- `fridge_tianshi/reference/fridge_tianshi_main.png`
- `chili_sis/expressions/chili_sis_angry.png`
- `story_templates/four_panel_comic.md`

## Character Naming Pattern

- Chinese display name: emotional and culturally flavorful.
- English asset name: stable production identifier.
- Avoid renaming core folders once assets exist, because prompts, scripts, and production references may depend on them.
