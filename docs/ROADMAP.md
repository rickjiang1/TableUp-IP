# TableUp Roadmap

## Phase 1: MVP

Target:

- Chinese users in the United States
- Practical household cooking
- Strong dinner-decision loop

Core goals:

- Reliable ingredient capture
- Inventory management
- Ingredient matching
- Unit conversion
- Storage-life rules
- Structured recipe library
- Can-cook and almost-there recommendation
- Basic household sharing
- Dev/prod environment separation

Recipe goal:

- Start with a smaller set of high-quality structured recipes.
- Long-term MVP target can be around 1000 curated recipes, but quality and structure matter more than count.

Knowledge base goals:

- Ingredient aliases
- Ingredient modifiers
- Taxonomy and tags
- Unit conversions
- Storage life
- Nutrition foundations

## Phase 2: Family Kitchen

Focus:

- Household inventory sharing
- Better member experience
- Invite flow polish
- Personal vs family inventory behavior
- Multi-device cloud sync

Recommendation improvements:

- Fridge Rescue Plan
- Smart expiration alerts
- Better almost-there logic
- Shopping-list-aware recipe suggestions

User memory:

- Dietary restrictions
- Disliked ingredients
- Common purchases
- Recipe history
- Waste patterns

## Phase 3: Pro Value

Potential paid features:

- Unlimited photo/receipt recognition
- Advanced Fridge Rescue Plan
- Weekly clear-fridge plan
- Smart grocery list
- Family Kitchen Sync
- Waste and savings reports
- Preference learning

Goal:

```text
Waste less food every week.
```

## Phase 4: Broader US Market

Focus:

- Western recipe library
- Modern Kitchen theme
- More English-first flows
- Broader ingredient data coverage
- More mainstream household onboarding

## Phase 5: International Expansion

Focus:

- Regional ingredient systems
- Regional recipe sets
- More language support
- Country-specific storage and grocery behavior

## Not A Priority Yet

Avoid over-investing early in:

- Huge generic recipe database
- Social/community features
- AI chat chef as the main interface
- Complex weekly meal planning before tonight recommendations are trusted

## Guiding Principle

TableUp should first become excellent at:

```text
What do I have, and what should I cook tonight?
```

Then expand into:

```text
How does my household waste less and plan better over time?
```
