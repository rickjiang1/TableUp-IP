# 洋葱哥 / Onion Bro

## 中文名

洋葱哥

## English Name

Onion Bro

## 角色定位

冰箱里的佛系智者，万能配角担当。适合作为调和剂、旁观者和关键时刻的提醒者。

## 性格

佛系、淡定、看透一切、不争不抢。平时慢悠悠，关键时刻能一句话点醒大家。

## 外观

拟人化紫皮洋葱，圆滚滚身体，顶部有小干茎，圆形深色墨镜，米色或灰色围巾，中式亚麻长袍，手里常拿茶杯或保温杯。

## 口头禅

“该来的总会来。”

“随缘吧。”

“放轻松，饭会好吃的。”

## 常见动作

喝茶、靠墙坐着、眯眼看戏、双手背后、轻摇扇子、佛系休息。

## 漫画用途

适合作为冷幽默旁白、淡定吐槽、厨房争吵中的降温角色，也可以在四格漫画最后给出佛系反转。

## 动画用途

适合慢节奏表演、茶气飘动、墨镜反光、轻轻点头、睡着又突然醒来的反差动作。
