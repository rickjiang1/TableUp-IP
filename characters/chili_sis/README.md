# 辣椒姐 / Chili Sis

## 中文名

辣椒姐

## English Name

Chili Sis

## 角色定位

辣味担当；冰箱江湖里的火辣核心、爆发型队友和气氛点燃者。

## 性格

自信张扬，敢爱敢恨，直率嘴亮，嘴甜心辣。
她看似强势不服输，实际非常重情重义，愿意为伙伴两肋插刀。
她的核心情绪是“热血不退”：越是遇到挑战，越要用行动把全场点燃。

## 外观

拟人化红辣椒，修长辣椒身姿，火辣曲线，火辣眼神，黑色短外衣，红色腰带，金色大耳环，黑色高跟短靴。
新形象带有黑红火焰纹头巾，整体更飒、更利落，也更适合江湖冒险和动画动作戏。

## 口头禅

“辣到你服，才算我赢！”

“来啊，怕辣就眨眼！”

“跟姐混，保证热血沸腾！”

## 常见动作

单手指向前方，叉腰挑眉，快速突进，甩出火辣爆弹，点燃火焰气场，得意眨眼，冲锋时留下红色动势。

适合制造高能入场、冲突升级、热血反击、辣味爆点和队友士气提升。她可以把平淡剧情瞬间点燃，也能在关键战斗中用“辣”打出情绪高潮。
