# 辣椒姐 / Chili Sis

## 中文名

辣椒姐

## English Name

Chili Sis

## 角色定位

火辣担当；提升食欲和情绪能量的气氛担当

## 性格

热情直爽，敢说敢做，爱憎分明，外表火辣内心温暖。

## 外观

拟人化红辣椒，绿色椒蒂，火辣眼神，金色耳环，黑色皮衣，红色腰带，黑靴。

## 口头禅

“生活就要辣得够味！”

## 常见动作

竖大拇指，点燃小火苗，叉腰训话，冲鸭式指挥，眨眼调味。
