# 姜姐 / Ginger Sis

## 中文名

姜姐

## English Name

Ginger Sis

## 角色定位

暖场指挥；去腥增香和行动派担当

## 性格

干练爽快，嘴上犀利但心很热，擅长把混乱变成步骤。

## 外观

拟人化生姜，姜黄色身体，弯曲有力的姜块轮廓，围裙或短外套，表情明亮坚定，可配木勺或小刀。

## 口头禅

“别慌，先把香味立起来。”

## 常见动作

拍案定流程，挥木勺指挥，切姜片，给队友鼓劲，闻味判断火候。

适合在队伍犹豫时推进执行，也适合厨房技巧提示。

## 动画用途

适合利落转身、姜片飞出、香气线条、快速切配节奏。
