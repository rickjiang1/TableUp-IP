# 豆小白 / Dou Xiaobai

## 中文名

豆小白

## English Name

Dou Xiaobai

## 角色定位

豆腐担当；冰箱江湖里的嫩白承味手、温柔辅助型伙伴和团队里的稳定承接者。

## 性格

外表柔软，内心善良，有点怕生但很认真。
豆小白说话轻轻的，遇到大场面会紧张，但关键时刻不会掉链子。
他不喜欢抢风头，却总能把大家的味道接住、融合好，是团队里让人安心的温柔力量。

他的核心情绪是“柔软但可靠”：看起来容易被碰碎，实际上很有承受力，也愿意替伙伴分担压力。

## 外观

拟人化嫩豆腐，方形白嫩豆腐身体，圆润大眼睛，柔和眉毛，脸颊微红。
头上系着清新的浅绿色蝴蝶结，穿绿色围裙和围巾，胸前有“豆”字圆牌。
整体气质软糯、清新、治愈，适合在江湖冒险中承担可爱、温暖和承味的角色。

## 口头禅

“别急，味道会进来的。”

“我虽然软，但我很能装味！”

“放心吧，交给我就对了！”

## 常见动作

招手邀请，双手捧碗，有点紧张地捏围巾，吸收汤汁时露出幸福表情，关键时刻鼓起勇气举手，认真站稳说“我来扛”。
