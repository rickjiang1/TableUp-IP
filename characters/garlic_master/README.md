# 大蒜爷 / Garlic Master

## 中文名

大蒜爷

## English Name

Garlic Master

## 角色定位

隐世高手；调味守护者与厨房老前辈

## 性格

沉默寡言，慢热，稳重，关键时刻准狠发言。

## 外观

拟人化大蒜头，蒜瓣纹理明显，白胡子，圆墨镜，灰色围巾与长袍，手持茶杯。

## 口头禅

“别吵。”

## 常见动作

端茶、捋胡子、闭眼沉思、轻轻抬指、从围巾里拿出调味建议。
