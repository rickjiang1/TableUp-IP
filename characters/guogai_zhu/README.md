# 锅盖猪 / Guogai Zhu

## 中文名

锅盖猪

## English Name

Guogai Zhu

## 角色定位

猪肉担当；冰箱江湖里最能打的回锅肉伙伴，锅盖为盾、锅铲为剑，是冲锋型肉盾和热锅气氛制造者。

## 性格

胆大心粗，话多先行，容易上头，但心地不坏。
他爱热闹、讲义气，遇到危险会第一个冲出去挡在伙伴前面。
虽然常常看起来莽撞，其实特别在意大家有没有吃饱、有没有被欺负。

他的核心情绪是“越战越香”：越遇到压力，越能把热锅气、肉香和勇气一起爆出来。

## 外观

拟人化猪肉侠客，圆胖结实的猪身，挑翻大耳，爆裂鼻孔，短粗腿。
头顶锅盖头盔，手持写着“炒”的木铲，另一手拿铁锅盖当盾牌，腰间有布围裙和酒葫芦。
整体形象粗犷、好笑、能打，像一个从市井小厨冲进江湖战场的回锅肉战士。

## 口头禅

“锅在手，肉我有！”

“干就完了！”

“别拦我！先干了再说！”

## 常见动作

举锅盖防御，挥木铲冲锋，大笑拍肚皮，闻到肉香鼻孔喷气，慌张时抱锅盖后退，上头时一边喊一边往前冲。
