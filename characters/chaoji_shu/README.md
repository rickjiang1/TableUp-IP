# 炒鸡叔 / Chaoji Shu

## 中文名

炒鸡叔

## English Name

Chaoji Shu

## 角色定位

街厨担当；冰箱江湖里的烟火喷嚏者、快乐街厨和爆炒气氛发动机。

## 性格

热情豪爽，幽默乐观，见人就招呼，见锅就来劲。
炒鸡叔不怕场面乱，反而越热闹越兴奋；他相信没有什么烦恼是一锅爆炒解决不了的。
他看起来夸张又爱表现，但本质上很会照顾大家的胃口和情绪，是团队里的开心火力和厨房节奏担当。

他的核心情绪是“锅气一到，烦恼全跑”：用笑声、火候和香气把低落场面重新炒热。

## 外观

拟人化街厨公鸡，橙红鸡冠像火焰，戴写着“炒就完了”的头巾，橙色护目墨镜，白色厨衣，橙色围裙，腰间挂着调料瓶和小物件。
一手持大锅铲，一手端着翻飞锅气的炒锅，脚爪健壮，整体充满街头烟火感和江湖厨师气势。

## 口头禅

“锅气一到，烦恼全跑！来，整两下！”

“没什么是一锅解决不了的！”

“不行？那就再加把火！”

## 常见动作

举锅铲大笑，双拳欢呼，翻锅爆炒，锅气冲天，戴墨镜挑眉，端锅冲进场，挑战时握拳喊话。
